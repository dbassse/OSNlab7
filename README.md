
##  🐸Нейронные сети для обработки временных рядов🐸

### 1. Предварительная обработка данных
* Очистка данных от пропусков (`NaN`) и некорректных форматов (удаление разделителей тысяч).
* Нормализация данных с использованием `StandardScaler` и `MinMaxScaler`.
* Создание выборок методом скользящего окна с помощью `TimeseriesGenerator` (Keras) и кастомных классов `Dataset` (PyTorch).

### 2. Архитектуры моделей
В проекте реализованы и протестированы следующие типы сетей:
* **Dense (DNN):** Полносвязные сети для быстрого базового прогноза.
* **Conv1D (CNN):** Сверточные сети для извлечения локальных паттернов в динамике цен.
* **LSTM (RNN):** Рекуррентные сети для учета долгосрочных зависимостей и сезонности.

### 3. уровень Pro
Для улучшения точности прогноза в данные были добавлены:
* Попарные разности и произведения ценовых каналов (Open, Close, Max, Min).
* Первые и вторые производные (скорость и ускорение изменения цены).
* Обратные значения признаков для выявления нелинейных зависимостей.

### 4. Оценка результатов
* **Регрессия:** Использование метрик MSE (Mean Squared Error) и MAE (Mean Absolute Error).
* **Визуализация:** Сравнение оригинального сигнала и прогноза на разных горизонтах (от 1 до 10 шагов вперед).
* **Статистический анализ:** Сравнение автокорреляционных функций (ACF) для проверки сохранения ритмичности сигнала.
